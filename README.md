Senior **Cybersecurity Engineer** based in 🇨🇦 Canada, specializing in **offensive security**, **red team operations**, and **enterprise security consulting**. I design and execute adversary simulations to help organizations identify critical vulnerabilities before real attackers do.

- 🔴 **Red Team** C2 infrastructure, EDR/AV evasion, lateral movement, Active Directory attacks

- 💼 **Financial sector focus** Risk prevention & enterprise security consulting
